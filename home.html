🧠 What is Memory Master?

This is a simple memory card game built using HTML, CSS and JavaScript.

🎯 Game Purpose

The game is designed to improve short-term memory and concentration. Each difficulty level trains your brain a little harder and see how fast you can recall where every card is!

⚡ The 3-Flip System

Unlike most memory games, you get to flip 3 cards per turn (Easy / Medium / Hard). If any two of those three match, they stay revealed and the third flips back. This makes the game faster, friendlier and more interesting!

🏆 Leaderboard

All scores are saved locally on your device. Check the Leaderboard to see top scores across all players and difficulties. Can you reach #1?

🛠 Tech Stack

Built with HTML, CSS and JavaScript. No frameworks. Data saved using browser LocalStorage. Fonts from Google Fonts.
