🧠 About Memory Master

A classic card-matching memory game built with HTML, CSS and JavaScript. No frameworks, no backend — just pure fun!

🎯 Goal

Train your short-term memory by finding all matching pairs as fast and accurately as possible.

🏆 Scoring

Score = (time left × 12 + accuracy bonus) × difficulty multiplier. Fewer moves + more time = better score.
